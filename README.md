A full-stack web application built using ASP.NET MVC, Entity Framework, and a MySQL database (managed via phpMyAdmin).
The project represents my first complete website built with the MVC architecture — focusing on clean structure, maintainability, and real-world CRUD functionality.
Key Features

ASP.NET MVC architecture — clear separation of concerns

Entity Framework ORM — database modeling, migrations, and LINQ queries

MySQL database configured and maintained through phpMyAdmin

User-friendly UI built with clean HTML + CSS

Fitness-oriented pages (exercises, programs, progress tracking)

Full CRUD support for core entities

Strong focus on code clarity and project structure
